Task:
Given a text file containing a list of employees and their attributes, print the list
sorted alphabetically by last name in the following format:

    Lastname, Firstname Title PhoneNumber

with PhoneNumber in the format (xxx)xxx-xxxx.


Solution:
I wrote a class called Employee to hold information for individual employees and a class
called EmployeeCollection which is a container of Employee objects and has a method to
print the list in the required format.


Instructions to run:
execute the command "source run.sh"
This script compiles example.cc and runs the executable.
